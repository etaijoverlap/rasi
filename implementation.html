rasi.rates¶

The rates module collects classes that implement different defect models. Currently, there is only the RateCalculator4State, which implements the current “workhorse”, the four state defect. The purpose of the calculators in this module is to collect together the rates for all microscopic transitions which are then used to compute measurable quantities or the transient behavior of defects.

class rasi.rates.RateCalculator4State(**kwargs)[source]¶

Calculates the microscopic rates for a four state defect.

Input

Variables:
  • recharge_primary – NMP rate calculator for the primary configuration
  • recharge_secondary – NMP rate calculator for the secondary configuration
  • nu – Attempt frequency
  • energies – Energy barriers
  • temperature – Temperature

Output

Variables:
  • rates – A dictionary containing the rates of the four state defect model
  • timeconstants – A dictionary containing the time constants (inverse of the rates)
